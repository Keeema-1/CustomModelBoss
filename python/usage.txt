
＜ボステンプレート作成プログラム＞

※注意事項※
このプログラムは自分用のものなので、他人に使ってもらえるほど整備されていません。
これらのプログラムは自由に書き換えて頂いて構わないので、自分用のものを作ってください。
使う際は自己責任でお願いいたします。

【使い方】

1. boss_settingsディレクトリ内にボス設定ファイルを作成します。usage.txtを参考にして、
   'そのボスを表す名前.json'として作成してください。

2. general_settings.pyのBOSS=''の部分の右辺に、2で作成したボス設定ファイルの名前
   (拡張子は含めない)を入力します。

3. コマンドプロンプトでこのファイルの存在するディレクトリに移動し、以下のように
   テンプレート作成プログラムを実行します。
python template.py

4. 実行するとデータパックにボスのテンプレートが作成されます。以下のコマンドを実行して
   開発用の本を受け取ることができます。
/function kboss:command/book/ボス名
